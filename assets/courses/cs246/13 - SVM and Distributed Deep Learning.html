SVM:

Assume a linear kernel for the purpose of this exercise.
Note: Though the line is written w⋅x+b, we will 1-pad x and just write w⋅x. The margin is then (w||w||⋅x) (1)
We wish to find a line w⋅x that best maximizes the margin mi between the two classes.

  • prediction for point xi: yi=sign(w⋅x)∈{−1,+1}
  • margin ("confidence"): mi=(w||w||⋅x)yi. (2)

Problem:

Given a dataset {xi}, we wish to classify each point to a label yi∈{−1,+1}. We wish to find parameters w such that the line w⋅x separates th two classes. So we want to solve maxw,bminimi, which we rewrite as

maxw,mms.t.∀i,yi(w||w||⋅xi)≥m
(Eq 1)

Given support vectors x1, x2, we have

  • x1=x2+2mw||w||
  • w⋅x1=+1, w⋅x2=−1,
    Therefore, w(x2+2mw⋅w||w||)=+1,
    and thus −1+2mw||w||=+1→m=||w||w⋅w=1||w|| (Picture in 2)

maxw1||w||s.t.∀i,yi(w||w||⋅xi)≥1||w||
(Eq 2)

And for the sake of training convenience, we change our objective function maxw1||w|| to minw||w|| to minw12||w||2.

minw12||w||2s.t.∀i,yi(w⋅xi)≥1
(Eq 3 - SVM with Hard Constraints)

Next, we introduce slack variables ζi to allow for imperfect separability.

minw,ζi≥012||w||2+C⋅∑i=1nζis.t.∀i,yi(w⋅xi)≥1−ζi
(Eq 4 - SVM with Slack Variables)

ζi is a penality term. If xi is on the wrong side of the margin, then get penalty ζi. (If margin ≥ 1, don't care, if margin < 1, pay linear penalty). We need to subtract the slack term from the constriants to make sure that they can still bind. C is a regularization term: very large C means misclassified points are heavily penalized - we want close to perfect separability. C = 0 means that we can basically set ζi to anything and there is a very low penalty - this is too slack and basically ignores the data.

Finally, we rewrite the above in its hinge loss form:

minw12w⋅w+C⋅∑i=1nmax{0,1−yi(w⋅xi)}
(Eq 5 - SVM with Hinge Loss)
Where each of the conditions yi(w⋅xi)>1−ζi became ζi>1−yi(w⋅xi), so it's min value for each i would be 1−yi(w⋅xi) if the point was misclassified, or 0 if the point was correctly classified.
