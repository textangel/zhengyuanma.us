Matrix Formulation of Simple Linear Regression

We can write the simple linear regression equation as a matrix equation.

Yn×1=Xn×iβi×1+ϵn×1

The objective function to minimize is then the square difference (Y−Xβ)T(Y−Xβ)=YTY−2βTXTY+βTXTXβ.
δ/δβ:−2XTY+2XTXβ=0
This gives us for closed form formula β̂ =(XTX)−1XTY.

Aside: Outlier Analysis and Diagnostics
Outlier Analysis and Diagnostics is really important to linear regression, because linear regression is highly sensitive to outliers. There is a alternate regression, robust regression, which finds outliers: minβ0,β1∑ni=1|Yi−β0−β1xi|. This is a convex program. There are other robust regression techniques also.

Logistic Regression

Simple Linear Regression can be applied to other distributions. For example,in logistic regression, we have
Given Yi∼indepBernoulli(p)f(pi)=pyi(1−p)1−yi;yi=0,1
We model pi using the logistic function pi=eβ0+β1xi1+eβ0+β1xi.
L(β0,β1)=∑iYilogpi+(1−yi)log(1−pi)

We can use the Newton-Ralphson method to update:
Let β=[β0,β1]T. We initialize β(0)=[0,0]T, and perform the update step:

  • β(k+1)=β(k)+(He(k))−1∇(k)
    until convergence.
