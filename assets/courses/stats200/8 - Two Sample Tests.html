(Chapter 11 in Book)
Two Sample Tests (Comparing a control group and a test group) are divided into

  • unpaired samples (normal case and the non-parametric case)
    • For normal case, a pooled standard deviation and a t-test.
    • For nonparametric case, we use the Mann-Whitney test.
  • paired samples (normal case and the non-parametric case)
    • Use a t-test as the unpaired case, but use σD=(1−ρ)‾‾‾‾‾‾‾√σ.
    • For the non-parametric paired case, use the W+ test statistic.

Given control group x1,...,xn, with a statistic μx,
we compute on the test group Y1,...,Ym the same statistic μy=μx+δ. A hypothesis test is whether this δ is 0 or not.

Unpaired Two Sample Test

Difference Between Two Means in Unpaired Samples
Treatment: X1,...Xn∼F
Control: Y1,...Ym∼G;F indep. G

Example:
Xi∼M(μx,σ2)
Yi∼M(μy,σ2). Note that we assume F and G have the same variance.
H0:μx=μy
H1:μx≠μy

We're going to use the test statistic X¯−Y¯. Then since
Var(X¯−Y¯)=Var(X¯)+Var(Y¯)−2Cov(X¯,Y¯)=σ2n+σ2m,
We have that X¯−Y¯∼N(μx−μy,σ2(1n−1m)).

So we can use the statistic z=X¯−Y¯σ1n+1m√ to do our hypothesis testing against N(0,1). If we are doing a one sided test this is straightforward. If we are doing a two sided test (H1:μx≠μy), we have to use the generalized likelihood ratio, which from the last lectures was −2logΛ=(X¯−μ0σ/n√)2∼χ21, so we test against

−2logΛ=(X¯−Y¯σ1n+1m‾‾‾‾‾‾‾√)2>χ21,α

However, in practice this is nearly equivalent to
∣∣∣∣∣X¯−Y¯σ1n+1m‾‾‾‾‾‾‾√∣∣∣∣∣>zα/2

with a caveat - When we use the sample standard devication instead of the population standard deviation, this becomes a t distribution.

Moving from normal to t dustributions:
We have the sample variance:
σ̂ 2=1n−1∑ni=1(Xi−X¯)2
(n−1)σ̂ 2σ2=1σ2∑ni=1(Xi−X¯)2∼χ2n−1
Cov(X¯,Xi−X¯)=0⟹ indep means that X¯∼N(μ,σ2n) and (n−1)σ̂ 2σ2∼χ2n−1 are independent.
Based on this knowledge we can write that
X¯−μσ̂ /n√∼tn−1.

Usually the variance estimator we use in the two means case is the pooled estimator for variance:

s2p=∑ni=1(Xi−X¯)2+∑mi=1(Yi−Y¯)2n+m−2

where
(n+m−2)s2pσ2∼χ2m+1−2

Putting it all together, for the difference between two means hypothesis test, we use
T=X¯−Y¯sp1n+1m‾‾‾‾‾‾‾√

Under H0:T∼N(0,1).
Under H1:μx−μy>0:T>tn+m−2,α
Under H1:μx−μy<0:T<tn+m−2,α
Under H1:μx−μy≠0:|T|>tn+m−2,α/2

Unpaired Two Sample Test for general, non-parametric distributions

Willcoxan Rank Sum Test (aka Mann-Whitney Test)

(p.438 of book)
The Willcoxan Rank Sum Test (aka Mann-Whitney Test) is a nonparametric test of the null hypothesis that the probability that two datasets come from the same generating distribution. More particularly, it tests that for randomly selected values X and Y from two populations, the probability that X is greater than Y is equal to the probability that Y is greater than X.

Suppose we have two datasets X and Y of ordinal values. Mix up the two datasets and rank their values from largest to smallest. (For example, if we labeled points in dataset X as X and points in Y as Y, the ranks might come out something like:
X Y X Y Y X Y Y Y
1 2 3 4 5 6 7 8 9

H0:F=G
H1:F≠G, where F, G are the distributions of X and Y.

Def: The rank sum statistic T is the sum over all of the X or Y datapoints' ranks.
TX=∑i∈XRi
TY=∑j∈YRj
Suppose we wanted to test H0:F=G,H0:F≠G, and suppose we know E[TY],Var[TY]. We would use

∣∣∣∣TY−E[TY]Var(TY)‾‾‾‾‾‾‾‾√∣∣∣∣>zα/2

as our test statistic.
Note: the is like sampling without replacement!
Note:
E[R¯]=μ1,Var(R¯)=(N−mN−1)σ2
μ=E[Ri]=1N∑Ni=1Ri=1NN(N+1)2=N+12
σ2=E[R2i]−E[Ri]2=1N∑Ni=1R2i−(N+1)2/22=1NN(N+1)(2N+1)6−(N+1)24

Ex)
X1,...,Xn∼F
Y1,...,Yn∼G
H0:F=G
H1:F≠G
Apply Mann-Whitney
TY=∑i:i∈YRi
Candidate intervals are TY−E[TY]Var(TY)√∼N(0,1).

Paired Two-Sample Tests

The only diffrence between this and the paired test is we use a different standard deviation σ0! This is to take into account the positive correlation between the two samples.
Paired tests have positive ρ so denominator is bigger, so variance is smaller, and the test has greater power for a given n.

Normal Case
A paired two sample test is used when Xi, Yi are not independent. (X1,Y1),(X2,Y2),...,(Xn,Yn). Example: Twin tests, tests in education with same student performing two tasks.

H0:μx−μy=0
H0:μx−μy≠0

Suppose:
Xi∼indN(μx,σ2)
Yi∼indN(μy,σ2)
Cor(Xi,Yi)=ρ

We call Di=Xi−Yi the difference between the pairs, Di∼indN(μx−μy,2σ2(1−ρ))

The variance σ2D:=2(1−ρ)σ2 for Di is derived as follows:
σ2D=Var(Xi−Yi)=Var(Xi)+Var(Yi)−2Cov(Xi,Yi)=σ2+σ2−2ρσ2=2(1−ρ)σ2
The test is D¯σD/n√=X¯−Y¯σD/n√∼N(0,1), that is, X¯−Y¯1−ρ√σ/n/2√∼N(0,1) under H0.
The only diffrence between this and the unpaired test statistic (X¯−Y¯σ/n/2√) is the 1−ρ‾‾‾‾‾√, where we use a different standard deviation such that σD.

Paired tests have positive ρ so denominator is bigger, so variance is smaller, and the test has greater power for a given n.

When we actually compute this, we have to use an estimate sD=1n−1∑ni=1(Di−D¯)2 for σD, and then out test is X¯−Y¯sD/n√∼tn−1 under H0, and note that the t has n−1 degree of freedom, as the pairing gave us an extra degree of freedom as compared to n−2 in unpaired.

Non Parametric Case
For this case, we compute the W+ test statistic, and the test is called the sign rank test.
For all the pairs i, we compute Di=Xi−Yi. We rank these value by their absolute value and also keep track of their signs.

Di abs value sign rank
D1=X1−Y1 ∥D1∥ sign1 R1
D2=X2−Y2 ∥D2∥ sign2 R2
D3=X3−Y3 ∥D3∥ sign3 R3
D4=X4−Y4 ∥D4∥ sign4 R4

Then we take only ranks corresponding to values with with positive signs and sum them. This is known as the W+ test statistic.
Under H0,
E[W+]=n(n+1)/4
Var[W+]=n(n+1)(2n+1)/24, and the test is
W+−E[W+]Var(W+)√∼N(0,1)
