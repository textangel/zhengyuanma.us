Movie Profit Prediction

Movie Profit Prediction

I performed varied feature generation and trained tree-based and SVM regression models to predict the financial success of movies using publicly available information.

Feature Generation

I employed various feature generation techniques including:
  • Training word embeddings on keywords extracted from movie overviews.
  • Movie Embeddings using PCA dimensionality reduced TF-IDF compositions of the keyword embeddings.
  • Running PageRank on actor-actor graph to generate actor popularity features.

Modeling Methodology

I ran the generated features through some sklearn regression pipelines using:
  • SVMRegressor and RandomForestRegressor as base models.
  • LASSO (L1) regularization for automated feature selection.
  • Beam search for hyperparameter tuning
  • Different subsets of input features for sensitivity analysis.

As a modeling constraint, it was important that the target column is uncorrelated with other metrics to begin with. In my modeling, I eventually discovered that profits and budgets seemed to have an exponential relationship, that is, log(profits) was linearly correlated with log(budgets). This led to the finding that log(profits/budgets) was not correlated with budgets at all, and was a suitable metric to predict for.

This project is meant as a showcase of statistical machine learning, model building, and data processing.
